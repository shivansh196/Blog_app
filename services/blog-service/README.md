# Blog Service

Handles blog post management for the platform.

## API Endpoints

### Posts
- `POST /blogs` - Create new post
- `GET /blogs` - List all posts
- `GET /blogs/:id` - Get single post
- `PUT /blogs/:id` - Update post
- `DELETE /blogs/:id` - Delete post

## Development

```bash
# Install dependencies
npm install

# Run in development
npm run dev

# Run tests
npm test
```
